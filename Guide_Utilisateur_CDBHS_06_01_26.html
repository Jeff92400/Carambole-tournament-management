1. Accueil (Dashboard)

Le tableau de bord offre une vue d'ensemble de l'activite du comite avec des alertes proactives et des statistiques detaillees.

1.1 Alertes de Relances

Le dashboard affiche un bandeau d'alertes pour les administrateurs indiquant les tournois necessitant une relance.

Fonctionnalites :

  • Affichage des tournois a venir necessitant une relance
  • Indication du nombre de jours restants avant chaque tournoi
  • Code couleur : orange si > 7 jours, rouge si <= 7 jours
  • Bouton "Relancer" pour acceder directement a l'emailing pre-rempli
  • Bouton vert pour marquer une relance comme deja envoyee
  • Affichage des relances deja envoyees avec la date d'envoi

1.2 Actions Rapides

Boutons d'acces rapide aux fonctions principales :

  • Competitions a jouer (bouton rouge - priorite principale)
  • Importer une competition (admin seulement)
  • Competitions jouees
  • Voir les Classements
  • Calendrier

1.3 Selecteur de Saison

Un selecteur permet de filtrer toutes les statistiques par saison (cycle Septembre-Aout).

1.4 Statistiques affichees

Statistiques globales :

  • Joueurs Actifs : Nombre de joueurs actifs dans la base
  • Competitions jouees : Nombre de competitions importees pour la saison
  • Participants cumules : Total des participations aux competitions de la saison

Inscriptions saison en cours :

  • Total des inscriptions
  • Nombre de convoques
  • Nombre de forfaits

Competitions saison en cours :

  • Total : Passes + A venir
  • A venir : Competitions programmees non encore jouees
  • Passes : Competitions deja jouees et importees

2. Competitions a Jouer

SECTION PRINCIPALE DE L'APPLICATION
Cette section permet de preparer et gerer les competitions a venir. C'est le coeur de l'application pour l'organisation des tournois.

2.1 Mise a jour des inscriptions CDBHS

Avant de generer les poules, il est essentiel de mettre a jour les donnees depuis le systeme CDBHS.

Important : Un bandeau d'avertissement s'affiche en haut de page indiquant la date du dernier import.

Procedure de mise a jour :

  1. Cliquez sur "Mettre a jour les inscriptions" dans le bandeau jaune
  2. Telechargez les fichiers CSV depuis le systeme CDBHS (Tournois et Inscriptions)
  3. Importez chaque fichier via le formulaire dedie (pages separees)
  4. Verifiez que les dates d'import sont actualisees

2.2 Generation des poules

Une fois les inscriptions a jour, vous pouvez generer les poules pour chaque competition.

Selection de la competition :

  • Choisissez le Mode de jeu (Libre, Bande, 3 Bandes, Cadre)
  • Selectionnez la Categorie (R1, R2, R3, etc.)
  • Cliquez sur "Charger les Inscrits"

Liste des inscrits - Le tableau affiche :

  • Nom/Prenom : Identite du joueur
  • Licence : Numero de licence FFB
  • Club : Club d'appartenance
  • Classement : Classement officiel du joueur
  • Convoque : Statut de convocation (case a cocher)
  • Forfait : Si le joueur a declare forfait

Gestion des convocations :

  • Cochez les joueurs a convoquer dans la colonne "Convoque"
  • Les modifications sont sauvegardees automatiquement
  • Le compteur de joueurs convoques s'actualise en temps reel

Generation des poules :

  1. Verifiez le nombre de joueurs convoques
  2. Selectionnez le nombre de poules souhaite
  3. L'algorithme repartit equitablement les joueurs par classement
  4. Les poules sont affichees avec la composition detaillee
Astuce : L'algorithme de repartition utilise un systeme de "serpent" pour equilibrer les niveaux entre les poules.

2.3 Feuilles de matches

Apres generation des poules, vous pouvez creer les feuilles de matches.

  • Feuilles individuelles : Une feuille par match avec les details des joueurs
  • Recapitulatif par poule : Vue d'ensemble de tous les matches d'une poule
  • Export PDF : Generation d'un document PDF pret a imprimer

2.4 Gestion des convocations

Les convocations peuvent etre envoyees par email depuis la section Emailing :

  1. Generez les poules et validez la composition
  2. Allez dans la section "Emailing"
  3. Selectionnez le type "Convocation"
  4. Les joueurs convoques sont automatiquement selectionnes
  5. Personnalisez le message si necessaire
  6. Envoyez les convocations

3. Competitions Jouees

Cette section liste toutes les competitions dont les resultats ont ete importes.

Informations affichees :

  • Date de la competition
  • Mode de jeu et categorie
  • Nombre de participants
  • Lieu
  • Actions (voir resultats, supprimer)

Import des resultats :

  1. Cliquez sur "Importer un Tournoi"
  2. Selectionnez le fichier Excel des resultats (format FFB)
  3. Verifiez les donnees importees
  4. Validez l'import
Bouton d'export disponible pour telecharger la liste des competitions.

4. Classements

Consultation des classements par mode de jeu et categorie.

Filtres disponibles :

  • Saison
  • Mode de jeu (Libre, Bande, 3 Bandes, Cadre)
  • Categorie (R1, R2, R3, etc.)
  • Club

Donnees du classement :

  • Position
  • Nom du joueur
  • Club
  • Points de match
  • Moyenne generale
  • Meilleure serie
  • Nombre de tournois joues

5. Calendrier

Le calendrier est accessible directement depuis la barre de navigation principale (visible pour tous les utilisateurs).

Le calendrier affiche toutes les competitions programmees pour la saison avec :

  • Vue mensuelle des competitions
  • Details : mode, categorie, lieu
  • Indication des competitions passees vs a venir

6. Statistiques

Analyse detaillee des donnees de la saison avec selecteur de saison.

Onglet Clubs

  • Podiums par Mode de Jeu : Classement des clubs par nombre de podiums (or, argent, bronze)
  • Clubs les Plus Actifs : Classement par nombre de participations

Onglet Joueurs

  • Joueurs les Plus Actifs : Top 10 par nombre de competitions
  • Meilleures Moyennes : Par categorie selectionnee
  • Meilleures Series : Par categorie selectionnee
  • Nouveaux Joueurs : Premiere participation dans la saison

Onglet General

  • Taux de participation par mode (inscriptions vs forfaits)
  • Repartition par categorie

7. Emailing

Systeme complet d'envoi d'emails aux joueurs avec plusieurs modes.

7.1 Composer

Email libre a un ou plusieurs destinataires.

  • Selection des contacts avec filtres (club, mode, categorie, activite)
  • Support des templates avec variables dynamiques
  • Option d'ajout d'images
  • Programmation d'envoi differe
  • Mode test pour validation

7.2 Relances

Systeme de relances automatisees pour les inscriptions aux tournois.

Types de relances :

  • T2 : Relance pour le 2eme tournoi qualificatif
  • T3 : Relance pour le 3eme tournoi qualificatif
  • Finale : Confirmation pour les finales departementales

Fonctionnalites :

  • Pre-selection automatique du tournoi depuis le dashboard
  • Templates personnalisables avec variables (nom, date, lieu, date limite)
  • Date limite d'inscription calculee automatiquement (tournoi - 7 jours)
  • Lieu du tournoi inclus dans le template
  • Suivi des relances envoyees

7.3 Resultats Tournoi

Envoi automatique des resultats aux participants apres un tournoi.

  • Selection du tournoi
  • Texte d'introduction et conclusion personnalisables
  • Substitution automatique des variables

7.4 Convocation Finale

Notification des finalistes qualifies avec details de l'evenement.

  • Position dans le classement
  • Details de la finale
  • Exigences de participation

7.5 Gestion Contacts

  • Synchronisation avec la base joueurs et inscriptions
  • Filtrage par statut d'activite, club, mode, categorie
  • Edition manuelle avec suivi opt-in
  • Indicateurs de validite email

7.6 Templates et Historique

  • Templates : Modeles d'emails reutilisables avec reinitialisation par defaut
  • Historique : Consultation des emails envoyes avec classification par type
  • Programmes : Gestion des emails planifies avec modification/annulation

7.7 Messages des Joueurs NOUVEAU

NOUVEAU dans la version 3.1
Les joueurs peuvent desormais envoyer des messages au CDBHS directement depuis l'Espace Joueur (application mobile).

Reception des messages :

Les messages envoyes par les joueurs arrivent dans la boite mail du CDBHS (cdbhs92@gmail.com) avec :

  • Sujet prefixe par "[Espace Joueur]"
  • Nom, prenom et licence du joueur
  • Club du joueur
  • Email du joueur (pour repondre directement)
  • Contenu du message

Sujets possibles :

  • Question generale
  • Probleme d'inscription
  • Probleme technique
  • Suggestion
  • Autre

Confirmation automatique :

Le joueur recoit automatiquement une copie de son message pour confirmer l'envoi.

Conseil : Utilisez la fonction "Repondre" de votre messagerie pour repondre directement au joueur - l'adresse de reponse est pre-configuree.

8. Fichiers (Administration)

Menu reserve aux administrateurs pour la gestion des donnees.

Sous-menus :

  • Joueurs : Liste et gestion des joueurs (avec bouton export)
  • Competitions jouees : Acces a la liste des tournois (avec bouton export)
  • Tournois CDBHS : Donnees des tournois importees
  • Inscriptions CDBHS : Donnees des inscriptions importees
  • Clubs : Gestion des clubs et alias
Les imports de Tournois et Inscriptions CDBHS sont sur des pages separees pour plus de clarte.

9. Parametres

Configuration de l'application.

9.1 Mon mot de passe

Changement du mot de passe personnel (disponible pour tous les utilisateurs).

9.2 Mes parametres

Configuration personnelle :

  • Email personnel : Utilise pour la reinitialisation du mot de passe
  • Alertes tournois : Recevoir des notifications email pour les tournois a venir

9.3 Gestion des Utilisateurs (Admin)

Gestion complete des comptes utilisateurs :

  • Ajouter un utilisateur (nom, email, mot de passe, role)
  • Roles : Administrateur (acces complet) ou Viewer (lecture seule)
  • Modifier l'email d'un utilisateur
  • Reinitialiser le mot de passe
  • Activer/desactiver un compte
  • Supprimer un utilisateur
  • Activer/desactiver les alertes tournois par utilisateur

9.4 Parametres des epreuves (Admin)

Configuration des regles pour chaque mode de jeu et categorie :

  • Coin (Grand Coin / Petit Coin)
  • Distance normale et reduite
  • Nombre de reprises
  • Moyennes qualificatives (mini/maxi)

9.5 Planification des emails (Admin)

Configuration de l'heure d'envoi automatique des emails programmes (heure de Paris).

9.6 Alias des Clubs (Admin)

Gestion des correspondances entre variantes de noms de clubs pour l'affichage correct des logos.

10. Connexion et Reinitialisation

Connexion

Acces a l'application via la page de connexion avec nom d'utilisateur et mot de passe.

Reinitialisation du mot de passe

En cas d'oubli du mot de passe, les utilisateurs peuvent le reinitialiser eux-memes :

  1. Cliquez sur "Mot de passe oublie ?" sur la page de connexion
  2. Entrez l'adresse email associee a votre compte
  3. Recevez un lien de reinitialisation par email
  4. Cliquez sur le lien et definissez un nouveau mot de passe
Prerequis : L'utilisateur doit avoir configure son email dans ses parametres (section 9.2).

11. Espace Joueur (Application Mobile) NOUVEAU

NOUVEAU dans la version 3.1
Une application mobile "Espace Joueur" est maintenant disponible pour les joueurs du CDBHS.

L'Espace Joueur permet aux joueurs de :

  • Consulter les competitions a venir
  • S'inscrire aux competitions de leur categorie
  • Suivre leurs inscriptions et convocations
  • Consulter leurs classements
  • Contacter le CDBHS

Inscription des joueurs :

Les joueurs doivent creer un compte avec :

  • Leur numero de licence FFB (qui doit etre dans la base de donnees)
  • Une adresse email
  • Un mot de passe securise

Inscriptions via l'application :

Lorsqu'un joueur s'inscrit a une competition via l'Espace Joueur :

  • L'inscription est automatiquement ajoutee a la base de donnees
  • Le joueur apparait dans la liste des inscrits de la competition
  • Un email de confirmation est envoye au joueur
Un guide utilisateur separe est disponible pour l'Espace Joueur : Guide_Utilisateur_Espace_Joueur_CDBHS.html
