🔔 Test Push Notifications

Envoyez des notifications de test à votre mobile avant le déploiement Phase 5.

💡 Exemples de notifications à tester :

  • Convocation: "Vous avez été convoqué pour 3 Bandes R1 du 21/03"
  • Relance: "Rappel : inscrivez-vous avant le 15/03 pour Libre N2"
  • Résultats: "Les résultats du tournoi Cadre 47/2 sont disponibles"
  • Annonce: "Nouvelle annonce importante du CDBHS"

🔍 Informations de débogage

ℹ️ Notes importantes

  • Assurez-vous d'avoir activé les notifications dans l'app Player
  • Sur iOS, l'app doit être ajoutée à l'écran d'accueil
  • La notification apparaîtra même si l'app est fermée
  • Cliquer sur la notification ouvrira l'app à l'URL spécifiée
  • Les notifications sont envoyées en temps réel (< 1 seconde)
